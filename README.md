サービス概要
·保存やダウンロードした画像をドット絵に変換させる
·変換させた画像は色を塗り替えたり等、好みに合わせカスタマイズができる

## このサービスへの思い・作りたい理由
SNSを利用し始めた際にオリジナリティのあるアイコンが設定したかった。
僕自身がドット絵が好きでアイコンに利用しようとしたこともあったが、同様のドット絵変換アプリを使用してだとドット数が多いと細かすぎて潰れることがあったりし、SNSによってどれぐらいのドット数が適しているかが分からなく調整する必要があった。
だからこそ、アイコン利用に特化させた変換アプリがあると便利だと思った。

## ユーザー層について
想定しているユーザー層:SNSを利用し始めた方
SNSを利用し始めた方を対象としている
顔ともなるアイコンの設定に悩んでる方や個性のあるアイコンを求めている方へ向けたアプリ

## サービスの利用イメージ
画像をアップロードすることで、ドット絵へ変換されます
変換時にピクセル数を変更したり、変換後にドット単位で色を変えたり等、自分好みにカスタマイズすることも可能です
完成した画像は保存して、SNSのプロフィールアイコンへ設定できます

手軽に、自分だけのアイコンを作ることを目的としたアプリケーションとなっています

## サービスの差別化ポイント・推しポイント
競合アプリ
1⃣ミニドット絵メーカー３
良い点:アプリ内にツールや機能の使い方·説明が書いてありアプリを初めて利用する際もわかりやすい
悪い点:アイコン制作を目的とした場合、SNSで設定してみるまでアイコンとしての完成度が把握しにくい

2⃣ドット絵ナニカ
良い点:画像をドット絵に変換した際にどの色を何個使用したかを表示してくれる
悪い点:機能が画像のドット絵変換のみのため変換後のドット絵をカスタマイズできない

推し·差別化ポイント
同様のアプリケーションと比べアイコンへの利用に特化させた設計となっている
・円形にプレビュー表示
  ⇒保存前にアイコンにした際にどのように切り取れるかの確認ができる
・ドット絵のドット数を32×32～256×256に選択でき、各ドット数ごとにSNSのアイコンを表示する
  ⇒ドット絵作成時からどのSNSへの利用に適したドット数かが把握できる

## 機能候補
MVPリリース時
・ドット絵への変換
・ドット毎への色塗り
・円形にプレビュー表示
・ドット絵の解像度の変更
・完成したドット絵の保存または、円形へトリミングし保存

本リリース時
・色フィルター
  ⇒ドット絵変換の際に「〇色系フィルター」「パステルフィルター」などの選択肢を指定することで変換後のドット絵に様々な雰囲気を持たせられる

・アカウントを作成しアプリ内でドット絵の保存、再編集をできるようにする

・パレット機能(使用した色を保存し再利用できるようにする)、スポイト機能
  ⇒使用する色をRGBで指定して使用できるようにしようと思っているため、別の色を使用した後でも再度RGBを設定しなくても過去に使用した色を使える
  また、別の作品を作る際には他作品で使用した色を持ち越して使用できる
