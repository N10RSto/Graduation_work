## サービス概要
·保存やダウンロードした画像をドット絵に変換させる
·変換させた画像は色を塗り替えたり等、好みに合わせカスタマイズができる

## このサービスへの思い・作りたい理由
僕自身が、何か自分を表せるものがなくツイッター等SNSのアイコンに何を設定するかを悩んでいました。
そこで、アイコンを作成できるサービスがあればいいのではないかと思い、僕が好きなゲームの要素と合わせてドット絵のアイコンが作成できるサービスを考えました。

## ユーザー層について
想定しているユーザー層:10～20代
SNSを利用し始めた方を対象としています
顔ともなるアイコンの設定に悩んでる方や個性のあるアイコンを求めている方へ向けたアプリです

## サービスの利用イメージ
画像をアップロードすることで、ドット絵へ変換されます
変換時にピクセル数を変更したり、変換後にドット単位で色を変えたり等、自分好みにカスタマイズすることも可能です
完成した画像は保存して、SNSのプロフィールアイコンへ設定できます

手軽に、自分だけのアイコンを作ることを目的としたアプリケーションとなっています

## ユーザーの獲得について
X内での宣伝が主になると考えています

## サービスの差別化ポイント・推しポイント
円形にプレビュー表示ができる
ドット絵の解像度を32×32～256×256に選択でき、各解像度ごとにどのSNSへのアイコン利用に適しているかを表示する

## 機能候補
MVPリリース時
・ドット絵への変換
・ドット毎への色塗り
・円形にプレビュー表示
・ドット絵の解像度の変更
・完成したドット絵の保存または、円形へトリミングし保存

本リリース時
・アカウントを作成しアプリ内でドット絵の保存、再編集をできるようにする
・パレット機能(使用した色を保存し再利用できるようにする)、スポイト機能
